HTML Structure

This theme is a fixed layout with 4+ columns. All of the information within the main content area is nested within a div with an id of "container". The Navbar (column #1). The header (column #2) content is within a div with an id of "home". The about (column #3) content is within a div with an id of "about". The services (column #4) content is within a div with an id of "service". The portfolio (column #5) content is within a div with an id of "Portfolio". The testimonials (column #6) content. The my-blog (column #7) content is within a div with an id of "blog". The contact (column #8) content is within a div with an id of "contact".The general template structure is the same throughout the template.





------ To Change the main color style ------


  • Change the file main.css to the required style
  • if you want the green style write green-style.css
  • if you want the blue style write blue-style.css

JavaScript

This theme imports three Javascript files.

  • jQuery
  • My custom scripts

jQuery is a Javascript library that greatly reduces the amount of code that you must write. In addition to the custom scripts, I implement a few "tried and true" plugins to create the effects. This plugin is packed, so you won't need to manually edit anything in the file.
